# Chapter 12 — Advanced Python 1

Modern Python features: walrus operator, type hints, pattern matching, exception handling, scope, and comprehensions.

## Concepts
- `01_walrus.py` — the walrus operator (`:=`) for assignment expressions
- `02_types.py` — type hints and annotations
- `03_match_case.py` — structural pattern matching (`match`/`case`, Python 3.10+)
- `04_dictionary_merge_operators.py` — merging dicts with `|` and `|=`
- `05_exceptions.py` — handling exceptions with `try`/`except`
- `06_raising_exceptions.py` — raising custom exceptions with `raise`
- `07_try_with_else.py` — using `else` in `try` blocks
- `08_try_with_finally.py` — the `finally` block for guaranteed cleanup
- `09_main.py` — the `if __name__ == "__main__"` pattern
- `10_global_keyword.py` — the `global` keyword and variable scope
- `11_enumerate.py` — using `enumerate()` for index-value loops
- `12_list_comprehensions.py` — concise list creation with comprehensions
- `module.py` — a helper module used by the `09_main.py` script

## Practice Problems
5 problems applying exception handling, comprehensions, and modern Python syntax.
