# Project 2 — The Perfect Guess

A command-line number guessing game where the player tries to guess a randomly chosen number between 1 and 100.

## How it works
- The computer picks a random number using Python's `random` module.
- The player enters guesses repeatedly until they get it right.
- After each wrong guess, the game tells the player to go higher or lower.
- Once correct, the total number of attempts is displayed.

## Run it
```bash
python3 main.py
```

## Concepts applied
`random` module, `while` loops, conditionals, and f-strings — drawing on Chapters 2, 6, and 7.
