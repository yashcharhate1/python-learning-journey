# Chapter 11 — Inheritance and More on OOP

Advanced OOP concepts: inheritance hierarchies, `super()`, class methods, decorators, and operator overloading.

## Concepts
- `01_inheritance.py` — single inheritance and method overriding
- `02_multiple_inheritance.py` — inheriting from more than one parent class
- `03_multilevel_inheritance.py` — chaining inheritance across multiple levels
- `04_super.py` — using `super()` to call parent class methods
- `05_class_method.py` — class methods using `@classmethod`
- `06_property_decorator.py` — using `@property` for controlled attribute access
- `07_operator_overloading.py` — customizing operators with dunder methods (e.g., `__add__`, `__str__`)

## Practice Problems
7 problems applying inheritance, `super()`, properties, and operator overloading.
