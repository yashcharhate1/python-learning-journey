## Structure

Each chapter folder contains:
- `Concepts/` — topic-wise scripts written while learning a concept
- `Practice-Problems/` — problems solved to reinforce that chapter's topics

```
Chapter-XX-Topic-Name/
├── Concepts/
└── Practice-Problems/
```

## How to run any file

```bash
python3 path/to/file.py
```

No external dependencies are required — everything uses Python's standard library.
