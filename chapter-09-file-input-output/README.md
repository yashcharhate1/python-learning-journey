# Chapter 9 — File Input/Output

Reading from and writing to files in Python using various modes and methods.

## Concepts
- `01_file.py` — opening and reading files
- `02_file_write.py` — writing to files using write mode
- `03_more_file_functions.py` — additional file handling methods (`seek`, `tell`, `read`, `readline`, `readlines`)
- `04_append.py` — appending to existing files without overwriting
- `05_files_using_with_function.py` — the `with` statement for safe file handling (auto-closes file)
- `file.txt`, `myfile.txt`, `my_append_file.txt` — sample files used by the concept scripts

## Practice Problems
11 problems covering file reading, writing, appending, copying, renaming, censoring text, logging, and more.

Sample files used by practice problems: `poem.txt`, `donkey.txt`, `log.txt`, `censor.txt`, `Hi-score.txt`, and others — included alongside the scripts that use them.
